## 학습 내용

* UI를 개발하는데 필요한 적절한 HTML 태그를 선택하고 사용한다.
* 시맨틱 태그를 이해하고 사용한다.
* CSS의 position, float, flex 속성을 사용해서 화면 레이아웃을 구성한다.
* CSS의 캐스캐이딩, 상속, box-model 개념을 이해한다.





## 요구사항 분석

* 상단 메뉴 표시
* 스크롤 내렸을 때 일부 header 요소에 display fixed 속성 추가
* 검색어 input 표시
* 메뉴 리스트 표시
* 슬라이드 배너 구현
* 컨텐츠 표시
* footer 표시
